# scaling-octo-happiness
---
*wifi brute force*

I had reset my Wi-Fi router and then I found that the factory sticker was destroyed. This Python script helps me to determine the correct last digit of the password.

![example](https://github.com/danilamblia/scaling-octo-happiness/blob/main/scsh2134.png)

# Usage

To use the script, you need:

- Python modules (pywifi, time)
- A "passwords.txt" file that contains a variety of passwords
